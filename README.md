## 5. PROJECT CONTEXT: BORDER TRACKER 🗺️

### Tech Stack
- **Framework**: Next.js 16 (App Router), React 19
- **Styling**: Tailwind CSS v4 (Mobile-First, Glassmorphism)
- **Icons**: Lucide React + Framer Motion
- **Mapping**: Leaflet + React Leaflet (static markers, SSR-disabled)
